Get(string, string, string, string, string, string, string, string)

Gets HtmlString Icon string from the user manager resource file. Everything has a default, named: params recommended.

public static IHtmlString Get(string iconName = "search", string iconSize = "md", string iconSet = "Google", string iconTag = "i", string wrapperTag = "none", string styleColor = "none", string styleMargin = "0 0.5rem 0 0", string styles = "")

Parameters

iconName string

The key/name of the icon key to get.

iconSize string

xs, sm, md (default), lg, xl.

iconSet string

The public icon setlocalization key to get (with or w/o file ext).

iconTag string

Default is <I>, but allows <SPAN> or others to be specified

wrapperTag string

An HTML tag to wrap the output in

styleColor string

Set the foreground color using inline style. Any valid CSS style color syntax including names; https://developer.mozilla.org/en-US/docs/Web/CSS/color

styleMargin string

Set Margins using inline style

styles string

Additional styles to add to the inline style attribute

Returns

IHtmlString

A HtmlString containing the Icon to display.

Remarks

for iconSize follow Tw/Bs conventions? xs, sm, md (default), lg, xl) see https://developers.google.com/fonts/docs/material_icons#styling_icons_in_material_design (see https://www.w3schools.com/icons/google_icons_intro.asp and others)
